1. Идентичность и логотип

Знак Платёж — кириллическая П, стилизованная как «платёжные ворота» (gate). Две стойки + перекладина.

Конструкция

Default (ink tile)
Inverted (paper tile)

ViewBox: 24 × 24

Tile: rx="4" (= borderRadius)

Posts: 3 × 14, x=4 / x=17, rx=".7"

Crossbar: 16 × 2.6, y=4.5, fill #fd6141 (rust)

Clear space: ≥ 3 units (1× post width)

Контексты использования

Платёж
Платёж Админ
Минимальный размер знака: 16px. При 16px стойки = ~2px, что обеспечивает читаемость на ретина-экранах. Для favicon используется тот же SVG.

Anti-FOUC: Inline-скрипт в index.html ставит data-theme на <html> до первого рендера React. CSS реагирует через html[data-theme='dark']. Переход между темами — CSS transition: background-color .2s ease, color .2s ease.

Принцип: Sider всегда тёмный

Сайдбар остаётся ink-чёрным в обеих темах. В light mode: #141210, в dark mode: #0a0806 (ещё темнее, чтобы отделиться от body #0f0d0a). Это создаёт якорь — навигация всегда стабильна.

11. Do / Don't

Ключевые принципы дизайн-системы.

Do Использовать token.colorTextSecondary и <Text type="secondary"> для вторичного текста. Они автоматически адаптируются к теме.
Don't Хардкодить цвета текста: color: rgba(26,24,20,.68). В тёмной теме будет нечитаемо.
Do Все машинные значения (ID, RRN, PAN, timestamp, latency) — JetBrains Mono. Все человеческие тексты — IBM Plex Sans.
Don't Смешивать шрифты: merchant name моноширинным или timestamp основным шрифтом.
Do font-variant-numeric: tabular-nums для всех числовых колонок. Суммы выровнены по правому краю.
Don't Пропорциональные цифры в таблицах — колонки «пляшут» при разных значениях.
Do Тёплые нейтралы везде. #f4f1ec paper, #0f0d0a body dark. Оба с лёгким yellow/brown undertone.
Don't Холодные серые (#f5f5f5, #fafafa, #1f1f1f). Они конфликтуют с тёплой палитрой.
Do borderRadius: 4px для контролов, 6px для карточек, 2px для тегов. Логотип: rx=4.
Don't Большие радиусы (8px+). Система визуально чёткая, не bubble-style.
Do Сайдбар всегда ink-dark. Он — визуальный якорь. Активный пункт — #fd6141 (rust) фон.
Don't Светлый сайдбар или сайдбар, меняющий цвет с темой. Он стабилен.
Do Drawer без маски (mask={false}). Оператор должен видеть таблицу за drawer'ом.
Don't Модалки для деталей транзакции / диспута. Только drawer — данные рядом с контекстом.
Do Countdown urgent (<24h): colorError + полупрозрачный фон. Warning (24-72h): colorWarning. Normal: secondary text.
Don't Единый стиль для всех дедлайнов. Оператор должен мгновенно различать urgency.
